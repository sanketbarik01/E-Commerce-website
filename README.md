# React Shopping Cart Application

## Project Overview
This project is a simple **ReactJS shopping cart application**.  
It displays a list of products and allows users to add items to a cart, update quantities, remove items, and view the total price dynamically.

The cart functionality is implemented using **React `useState`**.

---

## Objective
- To understand React component structure
- To manage state using `useState`
- To implement basic shopping cart features
- To build a simple and user-friendly UI

---

## Features
- Product list with:
  - Product name
  - Price
  - Image
  - Available stock (hard-coded)
- Add products to cart
- Increase and decrease product quantity
- Remove products from cart
- Display:
  - Quantity of each product
  - Price per unit
  - Total price per product
- Display overall cart total
- Checkout button with confirmation alert

---

## Technologies Used
- ReactJS
- JavaScript (ES6)
